一、概率论的基本概念

样本空间、随机事件

一般事件运算：交换律、结合律、分配律、德摩根律

概率、古典概型

1、频率
2、概率公理化定义
概率性质：
①P(B - A) = P(B) - p(AB)
②当A含于B，P(B - A) = P(B) - P(A) ; P(A) <= P(B)
③对任意两个事件AB：P(A ∪ B) = P(A) + P(B) - P(AB)
3、古典概型：有限个样本点
4、几何概型：无数个样本点

条件概率、全概率公式

1、条件概率
事件B已发生的条件下事件A发生的条件概率：p(A | B) = P(AB) / P(B) ( P(B) > 0 )

性质：
①对于任一事件A，有P(A | B ) >= 0
②P(Ω | B) = 1

2、乘法定理
设P(A) > 0，则有P(AB) = P(A)P(B | A)。易知，若P(B) > 0 ，则有P(AB) = P(B)P(A | B).

3、全概率公式和贝叶斯公式

全概率公式：
设B为样本空间Ω中任一事件，A1、A2、A3、… 、An为Ω的一个划分，且P(Ai) > 0 , i=1,2,3,…,n，则有P(B) = P(A1)P(B | A1) + P(A2)P(B | A2) + P(A3)P(B | A3) + … +P(An)P(B | An) = ∑P(Ai)P(B | Ai) (i从1到n)；

贝叶斯公式：设B为样本空间Ω中一事件，A1、A2、A3、… 、An为Ω的一个划分，且P(B) > 0，P(Ai) > 0，i=1,2,3,…,n，则有：
P(Ai | B) = [ P(B | Ai)P(Ai) ] / [ ∑P(B | Ai)P(Ai) ] （i=1,2,…,n）

独立性

1、事件的独立性
A、B相互独立（无关联），P(B | A) = P(B) ，P(AB) = P(A)P(B)
当P(AB) = P(A)P(B)，则称A、B是相互独立的
①定理1.4
若A、B相互独立，则 A与B’、A’与B、A’与B’ 也相互独立
②定理1.5
若事件A，B相互独立，且0 < P(A) < 1，则P(B | A) = P(B | A’) = P(B)
③若A、B既独立又互斥，那么A、B至少有一概率为0，当P(A) > 0 , P(B) > 0，A、B相互独立与A、B互不相容 不同时存在。
2、伯努利试验
若试验E只有两种可能结果：A与A’，则E称为伯努利试验，设P(E) = p , 0 < p < 1 ，此时P(A’) = 1-P ，将E独立地重复进行n次，则称这一串重复的独立试验为n重伯努利试验。

意思是n重伯努利试验中A出现k次的概率计算公式。
